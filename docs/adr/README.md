# ADR Index

This directory contains local Architecture Decision Records for Cost Console.
These ADRs document implementation-level decisions for this repository while
referencing `E:\Repositorios\platform-ai-architecture` for portfolio-wide
context.

## Convention

- File name format: `NNNN-short-title.md`.
- Numbering is sequential with four digits.
- Suggested statuses: `Proposed`, `Accepted`, `Superseded`, `Deprecated`,
  `Rejected`.
- Accepted ADRs should remain stable. If a decision changes, create a new ADR
  that supersedes the older one instead of rewriting history.
